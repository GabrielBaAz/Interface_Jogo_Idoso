# Interface_Jogo_Idoso
Trabalho em Rad com Python e tkinter
Este projeto tem como objetivo desenvolver uma interface de um jogo educativo voltado para idosos. O sistema deve registrar as tarefas realizadas e as pontuações de cada usuário, permitindo que os funcionários acompanhem o desempenho dos participantes. Eles querem que o programa se lembre das tarefas/pontuações que cada usuário já fez para os funcionários poderem saber quem está indo bem e salve o seu desempenho em um BD.
